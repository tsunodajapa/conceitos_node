## Conceitos estudados

- Métodos HTTP: POST, GET, PUT e DELETE
- Tipos de parâmetros: Query, Params, Body
- Middlewares e suas funcionalidades

## Requisitos de instalação
- npm >= 6.0.0
- node >= 10

## Como utilizar

### Servidor 

#### Execute o comando para instalações de dependências do servidor:

```
yarn
```

#### Iniciar o servidor:

```
yarn dev
```
